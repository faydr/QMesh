Detailed Description

This class implements the QMesh Frame. It provides both the storage of Frame data fields as well as various methods for performing various functionality needed for QMesh, such as:

  • Checksum (CRC) calculations
  • Time-to-Live (TTL) handling
  • FEC encoding/decoding
  • Serialization/deserialization

uint32_t Frame::calculateUniqueCrc ( void  )

Calculate the CRC for the "unique" information. The unique information consists of the type, stream_id, and payload. This method is primarily used to provide a unique "hash" for Frames in order to determine whether the QMesh node has seen them before.

◆ deserialize()

PKT_STATUS_ENUM Frame::deserialize ( shared_ptr< vector< uint8_t >>  buf)

Takes a vector of bytes and loads it into the Frame's internal data structures. This is designed to take bytes received off the air, so it performs FEC decoding, and will return a PKT_STATUS_ENUM with the results (testing stuff out):

  • PKT_FEC_FAIL – FEC decoding failed
  • PKT_BAD_SIZE – Packet size is inconsistent with bytes decoded
  • PKT_BAD_HDR_CRC – Bad packet header CRC
  • PKT_BAD_PLD_CRC – Bad packet payload CRC
  • PKT_OK – Packet decoded successfully
    Parameters
    bufshared_ptr to a vector of received, encoded bytes to decode

Detailed Description

Base class for Forward Error Correction. Provides some generically-useful functions, like interleaving, but otherwise just functions as a dummy FEC class.

Basic soft decoding. This soft decoding uses the whole-packet SNR to get a BER and thus fake some sort of soft decoding from this information. Should modestly improve the coding gain vs. hard decoding (roughly 0.2-0.3dB).
